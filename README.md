# Sure Diagnostics — Admin Console

Internal staff portal for managing patients, partners, and rebates.

**Live URL:** https://sure-admin.netlify.app

## Usage
Admin staff can:
- Add/edit/delete patient records
- Manage partner accounts and rebate rates
- Upload monthly CSV exports (auto-maps all columns)
- View system-wide KPIs and branch analytics

## Deployment
Connected to Netlify. Push to `main` branch → auto-deploys in ~30 seconds.
